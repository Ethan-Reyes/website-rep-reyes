## 🛠 Featured Engineering Projects

### 1. U.S. Aviation Data Dashboard (C# & Chart.js)

- **Data Engineering:** Developed a .NET ingestion engine to process large-scale FAA aviation datasets, resolving complex ETL issues involving shifted column indices and data normalization.
- **Visualization:** Engineered an interactive dashboard using **Chart.js**, featuring real-time distributions of aircraft types and engine configurations.
- **Problem Solving:** Successfully debugged cross-origin pathing and JavaScript runtime errors to ensure 100% data rendering accuracy on a static hosting environment.

### 2. 3D Visitor Globe (Three.js & Multi-Cloud)

- **Visual Engineering:** Built an interactive 3D globe using **Three.js** featuring a rotating Earth and amber atmospheric glow.
- **Architecture:** Implemented a multi-cloud bridge where **GitHub Pages** (Frontend) securely communicates with **Netlify Serverless Functions** (Backend) to plot real-time visitor coordinates via CORS-hardened endpoints.

### 3. AReS 3 Diagnostic Webpage (Embedded Systems)

- **Context:** Developed for **Textron Aviation** to support aircraft technicians.
- **Engineering:** Designed and deployed an internal technical reference application directly onto embedded AReS 3 hardware using **PowerShell** and **SSH**.

### 4. AI-Powered Transcription Pipeline (Python & OpenAI Whisper)

- **Automation:** Developed a custom Python pipeline using **Whisper** and **FFmpeg** to automate the transcription of 8+ hours of technical audio, programmatically formatting output into structured master documentation.

### 5. TDD Automated Test Suite (Jest & JSDOM)

- **Integrity:** 32+ tests across 6 suites covering UI logic and DOM integrity.
- **Security:** Engineered and validated **XSS (Cross-Site Scripting)** sanitization logic for secure data handling.

## 🌐 Cloud Architecture & Deployment

This portfolio demonstrates a modern **Serverless & Multi-Cloud** approach:

- **Frontend:** Hosted on **GitHub Pages** for high-speed, version-controlled delivery.
- **Backend:** Powered by **Netlify Functions** and **Netlify Blobs** for geolocation tracking and data persistence.
- **CI/CD:** Automated build pipelines with linear git history enforcement, ensuring a clean, professional production stream.
